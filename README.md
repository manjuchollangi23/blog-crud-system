# Blog CRUD System

## Overview

A Blog CRUD Application developed using PHP and MySQL with secure user authentication. Users can register, log in, and manage blog posts through Create, Read, Update, and Delete operations.

## Features

* User Registration
* User Login & Logout
* Password Hashing
* Session Management
* Create Blog Posts
* View Blog Posts
* Update Blog Posts
* Delete Blog Posts
* Responsive User Interface

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* XAMPP

## Project Structure

* index.php
* register.php
* login.php
* logout.php
* dashboard.php
* add_post.php
* edit_post.php
* delete_post.php
* db.php
* style.css

## How to Run

1. Start Apache and MySQL in XAMPP.
2. Create the database in phpMyAdmin.
3. Import the required tables.
4. Place the project folder inside `htdocs`.
5. Open:

   http://localhost/blog-crud/index.php
